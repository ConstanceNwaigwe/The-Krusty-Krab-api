**_Description_**

- An api server for if the Krusty Krab from the show Spongebob Squarepants had a website

---

**_Getting Started_**
- Git Clone branch
- run npm i
- run npm run dev
- View in http://localhost:2024
- You can also view the deployed api [here](https://the-krusty-krab-api.onrender.com/)
